Реализуйте функции для изменения размера изображения,
конвертации в черно-белое и применения фильтра Гаусса
(размытие).

Ознакомьтесь с функциями, которые предоставляют Pillow и
opencv-python для работы с изображениями. В Pillow вы
можете использовать метод resize() для изменения
размера, метод convert() для конвертации в черно-белое, а
также метод filter() с фильтром ImageFilter.GaussianBlur()
для размытия. В opencv-python используйте cv2.resize() для
изменения размера, cv2.cvtColor() для конвертации в
оттенки серого и cv2.GaussianBlur() для применения
фильтра Гаусса.
